# Rhythm Games Music Labeller

A hobby project designed to fetch songs' and music metadata from various sources like Remy, Silentblue, and other music databases, then populate and organize this metadata into song/music files.

## Purpose

This tool helps automate the process of gathering and applying metadata to music files used in rhythm games and personal music collections. It retrieves metadata from multiple sources and integrates it seamlessly into your music files.

## How to use

### How to bind songs

1. Search for the song you want to bind on a supported site such as https://remywiki.com/.
2. Copy the song URL from that site.
3. Add the URL to your binder configuration or the app's binding workflow.
4. Run the app to fetch and attach metadata to your song files.

More supported sources will be added in the future.
